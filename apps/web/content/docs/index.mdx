## Filosofia

- Monte sua própria stack: escolha só o que precisa, nada a mais.
- Templates mínimos: scaffolds enxutos, sem inchaço.
- Dependências atualizadas: sempre atuais e estáveis por padrão.
- Gratuito e open source: para sempre.

## Comece agora

### Pré-requisitos

- **Node.js LTS** - [Baixe em nodejs.org](https://nodejs.org/)
- **Git** (opcional) - [Baixe em git-scm.com](https://git-scm.com/) - se quiser inicializar um repositório git
- **Bun** (opcional) - [Baixe em bun.com](https://bun.com/) - se quiser usar o Bun como gerenciador de pacotes

### CLI (prompts)

```npm
npm create kubojs@latest
```

Siga os prompts interativos para escolher frontend, backend, banco de dados, ORM, camada de API e addons.

Pule os prompts e use a stack padrão:

```npm
npm create kubojs@latest --yes
```

### Stack Builder (UI)

- Acesse [/new](/new) para montar sua stack e copiar o comando gerado
- Ou abra via:

```npm
npm create kubojs@latest builder
```

### Agente de IA (plugin)

Deixe seu assistente de IA montar a stack por você. Instale o plugin kubojs e descreva o que quer em português (ou inglês) — ele planeja uma stack válida e gera o scaffold.

```bash
# Claude Code
/plugin marketplace add albuquerquesz/kubo
/plugin install kubojs@kubojs
```

Depois peça: _"create a fullstack app with Next, Hono, Postgres and Better Auth"_. Também funciona no Codex, opencode e em qualquer agente com MCP — veja [Usar dentro do seu agente de IA](/docs/cli/agent-workflows#use-it-inside-your-ai-agent).

## Configurações comuns

### Stack padrão

```npm
npm create kubojs@latest my-webapp \
  --frontend tanstack-router \
  --backend hono \
  --database sqlite \
  --orm drizzle \
  --auth better-auth \
  --addons turborepo
```

### Convex + React + Clerk

```npm
npm create kubojs@latest my-api \
  --frontend tanstack-router \
  --backend convex \
  --auth clerk
```

### App mobile (Expo)

```npm
npm create kubojs@latest my-native \
  --frontend native-uniwind \
  --backend hono \
  --database sqlite \
  --orm drizzle \
  --auth better-auth
```

### Monorepo vazio

```npm
npm create kubojs@latest my-workspace \
  --frontend none \
  --backend none
```

## Cola de flags

Veja a lista completa na [Referência da CLI](/docs/cli). Flags principais:

- `--frontend`: tanstack-router, react-router, tanstack-start, next, nuxt, svelte, solid, astro, native-bare, native-uniwind, native-unistyles, none
- `--backend`: hono, express, fastify, elysia, convex, self, none
- `--runtime`: bun, node, workers, none
- `--database`: sqlite, postgres, mysql, mongodb, none
- `--orm`: drizzle, prisma, mongoose, none
- `--api`: trpc, orpc, none
- `--auth`: better-auth, clerk, none
- `--payments`: none
- `--addons`: turborepo, nx, vite-plus, pwa, tauri, electrobun, biome, lefthook, husky, starlight, fumadocs, ultracite, oxlint, mcp, opentui, wxt, skills, evlog, none
- `--examples`: todo, ai, none
